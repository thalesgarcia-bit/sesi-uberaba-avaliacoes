Sobre o projeto
Sistema com Template Mestre integrado e edição visual estilo docs.

Esta versão do sistema adiciona um Template Mestre integrado ao ambiente principal. O administrador pode arrastar elementos, trocar fontes, subir a imagem oficial dos rodapés e cabeçalhos, definir espaçamentos, cores, linhas e publicar o padrão oficial para uso padronizado dos professores.
